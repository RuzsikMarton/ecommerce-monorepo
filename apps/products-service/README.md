# Products Service

Product catalog and management service.

## What it does
- Product CRUD operations
- Product search and filtering
- Category-based filtering
- Product sorting (newest, price)
- PostgreSQL integration with Prisma
- Syncs product updates via Kafka

## Tech Stack
- **Express** - Web framework
- **Prisma** - ORM
- **PostgreSQL** - Database
- **Clerk** - Authentication
- **KafkaJS** - Event streaming
- **CORS** - Cross-origin requests
